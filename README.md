# Online Dating App

コンピュータサイエンス学習プラットフォーム[Recursion](https://recursionist.io)の課題として作成したオンラインチャットアプリです。AI相手に簡単な会話を楽しめます。Vue.jsのVuexライブラリを使用しました。
